project name : bookNest

BookNest is an online book borrowing platform built for Assignment 8. It digitizes a library experience where users can browse books, filter categories, view private book details, borrow books, and manage their profile securely.



 liveLink: https://my-assignment-8.vercel.app;

 key features : 1.Responsive navbar with login/logout state
  2.Home page with banner, new arrivals, featured books, and extra sections
3.All Books page with search and category filter
4.Private book details page with image, description, quantity, and borrow toast
5. system with email/password and Google
6.Private profile page with update name and image
7.MongoDB for auth and data storage using env variables


TechStack : Next.js

Tailwind CSS
HeroUI
BetterAuth
